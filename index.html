This is a test page of the WeatherWidget

To work with the WeatherWidget, you need your openweather API service token.
If you don't have it, you need to go to https://home.openweathermap.org,
log in and get it. It's free.

To include a widget in your site's page, before initializing it,
place your token in the localStorage with the following command:

localSorage.setItem('WeatherWidgetToken', --Your token--)
